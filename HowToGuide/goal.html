Why did I create this guide?

I am undertaking a project based on ML and found some guides to be confusing in parts; so I wanted to make a fool-proof guide to the start up process so you can follow along with the notebooks in the fast.ai course.

My overall goal

My goal for this guide is to a simple and easy-to-follow tutorial for anyone who is seeking to learn about ML through fast.ai. If you feel I could improve this guide to more accurately acheive this goal, please contact me.
